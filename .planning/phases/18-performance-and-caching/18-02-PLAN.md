   ```typescript

   ```

3. Add a private helper method `_cachedFetchBody(uid: string, folder: string): Promise<ParsedMail>`:
   ```typescript
   private async _cachedFetchBody(uid: string, folder: string): Promise<ParsedMail> {
     const key = `${this.account.id}:${folder}:${uid}`;
     const cached = this.bodyCache.get(key);
     if (cached) return cached;
     const parsed = await this.imapClient.fetchMessageBody(uid, folder);
     this.bodyCache.set(key, parsed);
     return parsed;
   }
   ```

4. In `readEmail()` (line 118): replace `await this.imapClient.fetchMessageBody(uid, folder)` with `await this._cachedFetchBody(uid, folder)`.

5. In `downloadAttachment()` (line 188): replace `await this.imapClient.fetchMessageBody(uid, folder)` with `await this._cachedFetchBody(uid, folder)`.

6. Add public `invalidateBodyCache(folder: string, uid: string): void` method:
   ```typescript
   invalidateBodyCache(folder: string, uid: string): void {
     this.bodyCache.delete(`${this.account.id}:${folder}:${uid}`);
   }
   ```

Do NOT cache in `extractAttachmentText` — it delegates to `downloadAttachment` which already hits the cache. No double-caching needed.
