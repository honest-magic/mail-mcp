# Phase 19, Plan 01: Extract List-Unsubscribe Headers

## Objective

Extract RFC 2369 `List-Unsubscribe` and `List-Unsubscribe-Post` headers from parsed
email objects in `readEmail()` and surface them as structured lines in the header
block of the returned string. Enables AI agents to detect and act on mailing-list
unsubscription links without manual header inspection.

## Tasks

### Task 1 — Write failing tests for List-Unsubscribe extraction (TDD RED)

tdd="true"

**Behavior:**

`readEmail()` must append unsubscribe information to the header block when
`List-Unsubscribe` is present in parsed headers:

- Email with only an `https:` URL → `**Unsubscribe:** <url>`
- Email with only a `mailto:` URL → `**Unsubscribe (mailto):** <address>`
- Email with both → both lines appended (https first, mailto second)
- Email without the header → no unsubscribe lines in output
- One-click: when `List-Unsubscribe-Post: List-Unsubscribe=One-Click` is present,
  append `**Unsubscribe (one-click):** yes` after the https line

**Test file:** `src/services/mail.test.ts` (add a new `describe` block)

### Task 2 — Implement List-Unsubscribe extraction (TDD GREEN)

tdd="true"

**Implementation:**

In `readEmail()` in `src/services/mail.ts`, after the `Message-ID` block (line 185)
and before `header += \`\n---\n\n\``:

1. Call `parsed.headers.get('list-unsubscribe')` to get the raw header string.
2. Parse URLs from the angle-bracket format: extract all `<...>` tokens.
3. Separate https URLs from mailto URLs.
4. Append `**Unsubscribe:** <https-url>` for each https URL found.
5. Check `parsed.headers.get('list-unsubscribe-post')` — if it contains
   `List-Unsubscribe=One-Click`, append `**Unsubscribe (one-click):** yes`.
6. Append `**Unsubscribe (mailto):** <address>` for each mailto URL found (strip
   the `mailto:` scheme prefix so only the address is shown).

Add a private helper `parseUnsubscribeHeader(raw: string): { https: string[]; mailto: string[] }`
to keep `readEmail()` readable.

**Done criteria:** `npm test` passes — all new tests green, existing tests unaffected.
