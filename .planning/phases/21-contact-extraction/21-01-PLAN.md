# Phase 21 Plan 01: Extract Contacts Tool

## Objective

Implement the `extract_contacts` MCP tool that scans recent messages in a mailbox folder, aggregates sender addresses by frequency, and returns structured `ContactInfo` objects sorted by count descending.

### Task 2 — TDD GREEN: implement scanSenderEnvelopes in ImapClient
type="auto" tdd="true"

**Implementation:**

Add to `src/protocol/imap.ts`:
- Export `SenderEnvelope` interface: `{ name: string; email: string; date: Date }`
- Add `scanSenderEnvelopes(folder?: string, count?: number): Promise<SenderEnvelope[]>` to `ImapClient`
- Uses `this.client.fetch(range, { envelope: true })` — no bodyParts (lightweight)
- Normalizes email to lowercase; uses display name from envelope or empty string
- Skips messages where `from[0]?.address` is nullish

Tests must pass.

### Task 4 — TDD GREEN: implement extractContacts in MailService
type="auto" tdd="true"

**Implementation:**

Add to `src/services/mail.ts`:
- Export `ContactInfo` interface: `{ name: string; email: string; count: number; lastSeen: string }`
- Add `extractContacts(folder?: string, count?: number): Promise<ContactInfo[]>` to `MailService`
- Calls `this.imapClient.scanSenderEnvelopes(folder, count)`
- Aggregates by normalized email (lowercase)
- Tracks count + most recent date + name from most recent message
- Sorts by count descending, then by lastSeen descending as tiebreaker
- Returns at most 50 contacts

Tests must pass.
Commit: `feat(21-01): implement MailService.extractContacts`
